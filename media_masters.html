Media Masters

Event Coordinator

Padma Sidanewar
No. of Participants : 2

Both team members have to participate for all 3 events.

Event 1
Sync

Event Name

Sync

Description

A contest to synchronize a muted video clip with audio files to create a meaningful movie.

Rules
  • Participants will be provided with muted video clips and audio files.
  • The task is to synchronize any one of the video clips with one or more audio files to create a meaningful movie using Shortcut video editor / Openshot Video Editor.

Event 2
Morphos

Event Name

Morphos

Description

A contest to morph a set of given images to create a new image.

Rules
  • Participants will be provided with images.
  • Participants can use any of the images from the given set to create a new morphed image using GIMP/Photoshop editing software .

Event 3
CreaTIvITy

Event Name

CreaTIvITy

Description

A contest to design a poster based on a theme using a graphics editing software.

Rules
  • Participants are required to design a poster based on the given theme using GIMP/ Photoshop editing software .
