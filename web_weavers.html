Web Weavers

Event Coordinator

Sherwin Da Costa
No. of Participants : 2

Both team members have to participate for all 3 events.

Event 1
.html

Event Name

.html

Description

A web designing contest to create a static website based on a given theme using VSCode / Sublime text / Adobe Dreamweaver.

Rules
  • The participants will be given a theme and they have to design a static website.
  • Website should be designed using html, CSS and JavaScript.
  • Premade templates cannot be used.
  • Website will be judged based on originality, design, functionality and usability.

Event 2
<JS> Validator

Event Name

<JS> Validator

Description

A form validation contest to design and validate the given html form using VS Code/Sublime text /Adobe Dreamweaver.

Rules
  • The participants will be given a screenshot of the form layout.
  • The form should be designed using html, CSS and JavaScript.
  • Participants have to perform all possible JavaScript validations.
  • Use of constraint validation attributes eg. required, min, max will not be allowed.

Event 3
WebDev

Event Name

WebDev

Description

A web application testing contest to identify and correct the errors in a website.

Rules
  • The participant will be provided with a static website created with html, CSS and JavaScript. The website does not have server side programming and database connection.
  • The participants have to identify the errors, document the errors in the template provided, fix the errors and optimize the website for performance.
